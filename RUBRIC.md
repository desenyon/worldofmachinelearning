# Final Project Rubric (100 points)

## 1. Problem Framing (15)
- Clear user/problem statement
- Input/output contract documented
- Scope is realistic

## 2. Data Quality + EDA (20)
- Data source + license documented
- Cleaning decisions justified
- EDA insights are meaningful

## 3. Modeling + Metrics (25)
- Baseline + improved model shown
- Correct metric used for track
- Error analysis performed

## 4. Reproducibility (15)
- Setup instructions work
- Seeds/config captured
- Artifact paths are clear

## 5. Device Deployment (20)
- Model optimized and exported
- Local inference runs on target hardware
- Benchmark evidence included

## 6. Communication (5)
- README clarity
- Reflection quality

## Pass Threshold
- Minimum passing score: **85/100**
