# World of Machine Learning

A project-based machine learning program for motivated high school students, built by [Hack Club](https://hackclub.com).

![Hack Club](https://img.shields.io/badge/Hack%20Club-ec3750?style=for-the-badge) ![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=next.js&logoColor=white) ![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white) ![TailwindCSS](https://img.shields.io/badge/TailwindCSS-06B6D4?style=for-the-badge&logo=tailwindcss&logoColor=white)

## Overview

World of ML teaches students to build their first AI model end-to-end using real Kaggle datasets.
