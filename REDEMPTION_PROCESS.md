# Redemption Process

## Eligibility Rules
A learner must pass all gates:
1. Complete required Phase 1 lessons
2. Log minimum hours (`DEVICE_MIN_HOURS`, default 40)
3. Approved project meets metric threshold by track
4. Rubric score >= 85

## Workflow
1. Learner completes gates
2. Learner opens `/redeem`
3. Form unlocks automatically when eligible
4. Learner submits shipping details
5. Staff reviews and marks request status:
   - pending
   - approved
   - shipped
   - rejected

## Required Learner Evidence
- Repo URL
- Demo URL/video
- Model artifact path
- Build-time proof links

## SLA Guidelines
- Review turnaround target: 3-5 days
- Shipping window target: 2-4 weeks (region dependent)
